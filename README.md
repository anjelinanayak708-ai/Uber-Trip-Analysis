# Uber Trip Analysis Dashboard

## Overview

This project is an Uber Trip Analysis Dashboard developed using Power BI to analyze ride booking trends, trip distances, revenue generation, payment methods, vehicle performance, and location-based insights. The dashboard provides an interactive and visual representation of Uber trip data, helping users understand customer behavior, travel patterns, and operational performance more effectively.

## Dashboard Preview

![Uber Trip Analysis Dashboard](Dashboard%201.png)

## Features

The dashboard includes analysis of total bookings, booking revenue, average booking amount, total trip distance, average trip distance, and average trip duration. It also provides payment method analysis, day versus night trip comparison, vehicle-wise booking performance, and location-based pickup and drop-off analysis. Interactive filters for date and city selection allow users to explore the data dynamically.

## Key Metrics

The dashboard records a total of 103.7K bookings with a total booking amount of $1.6M. The average booking amount is 14.98, while the total trip distance covered is 349K miles. The average trip distance is 3 miles, and the average trip duration is 16 minutes.

## Insights

The analysis shows that Uber Pay is the most frequently used payment method, while cash and Amazon Pay are less commonly used. Day trips are higher compared to night trips, indicating greater demand during daytime hours. Among the vehicle categories, UberX has the highest number of bookings, while Uber Comfort and Uber Black also contribute significantly to revenue generation. The most frequent pickup point is Penn Station/Madison Sq West, and the most frequent drop-off point is Upper East Side North. The farthest recorded trip was from Lower East Side to Crown Heights North.

## Tools and Technologies Used

This project was developed using Power BI for data visualization and dashboard creation. Data modeling and DAX functions were used for calculations and KPI generation, while ETL and data cleaning processes were performed to prepare the dataset for analysis.

## Purpose of the Project

The primary objective of this project is to analyze Uber trip data efficiently and provide meaningful insights into customer travel behavior, booking trends, vehicle usage, and revenue generation. The dashboard helps in improving decision-making through visual analytics and interactive reporting.
